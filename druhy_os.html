Historie operačních systémů



Prvním operačním systémem byl CTSS (Compatible Time-Sharing System), který byl v roce 1961 vyvinut pro MIT. Tento systém podporoval sdílení času, e- mailovou službu a nacházel se zde zárodkem shellu (viz dále). Byl zde také přítomný program pro formátování textu.

O čtyři roky později vznik druhý OS - MULTICS (Multiplex Information and Computation Service), který byl ve své době poměrně neúspěšný, ale měl velký vliv na další vývoj operačních systémů.

Jeden z operačních systémů, který se MULTICSem inspiroval, byl UNIX, který vznikl roku 1970. Původně se jmenoval UNICS : Universal Computing Systém. Byl a dodnes je využíván na serverech, pracovních stanicích nebo i na osobních počítačích. Jeho výhodou je jednoduchost a zároveň poměrně velké možnosti.

MS-DOS, který v roce 1981 vytvořila firma Microsoft pro osobní počítače (zejména pak pro počítače od firmy IBM). Zkratka DOS je diskový operační systém (anglicky disc operating system). MS-DOS byl operační systém, jehož uživatelské rozhraní bylo textové, příkazy se zadávaly pomocí příkazového řádku. Mohl na něm probíhat pouze jeden program, každý pak měl jednu konkrétní funkci, každý měl jiné ovládání a pro práce se soubory se používaly nejrůznější nadstavby. Rozšíření osobních počítačů zapříčinil rozšíření tohoto OS. Postupně na něj vznikla grafická nadstavba ve formě prvního operačního systému s grafickým rozhraním. Operační systémy společnosti Microsoft, které jsou dnes nejrozšířenějším operačním systémem na osobních počítačích, právě jako jeden z prvních firem nabídl veřejnosti grafické rozhraní. Jméno OS Windows vzniklo díky práci s okny, která ve své době byla naprosto inovativní. Krátce po MS-DOSu nabídl Microsoft možnost pracovat s více procesy naráz tudíž již zmíněný multitasking. Postupně samozřejmě přidával na své operační systémy další vylepšení např. swapování (tj. rozšíření RAM pomocí HDD), podpora sítě peer-to-peer, sdílení zdrojů, podpora dlouhých jmen souborů, podpora plug-and-play,...

První “wokna“ se jmenovaly Windows 1.0, které vyšly roku 1985 a postupně začaly rychle přibývat další : Windows 2.0, Windows 3.0, Windows 3.1, Windows 3.5. Roku 1995 přišel nový OS s jiným číselným řazením a to Windows 95, který přinesl novou a výraznou změna v grafickém rozhraní a podporu plug-and-play. Dále to byl Windows 98 a poslední z této řady Windows Me (Milenium). Poté přišel na trh Windows 2000 následně Windows XP, který přinesl větší bezpečnost, odstranil staré chyby a vylepšil multimediální stránku systému. Po Xpčkách Microsoft vydal operační systém Windows Vista, který přes velké změny zejména opět v grafickém rozhraní nebyl veřejností příliš dobře přijat a když přišel nový operační systém Windows 7, tak svět opět začal Microsoftu věřit, že dokáže udělat inovativní systém. Po “sedmičkách“ přišly Windows 8, které hlavně kvůli úplně novému a uživatelům neznámému grafickému rozhraní (tzv. Metro) nebyly příliš kladně přijaty. Nyní probíhá beta verze zkoušení operačního systému Windows 10, které je zatím přijímáno s nadšením, protože kombinuje prvky metra a klasického oknového systému.

Dalšími operačními systémy jsou systémy rodiny Linux neboli GNU/Linux. Jsou to operační systémy z jádra UNIX (autorem je Linus Torvalds) a základních knihoven a nástrojů pocházejících z projektu GNU, jež byl založen Richardem Stallmanem. Linux je software, který je povětšinou zdarma (výjimkou je Red Hat Enterprise Linux) a může se předělávat podle potřeb uživatele tzv. open source system. Přestože Linux vychází svým jádrem z UNIXu tak s ním nemá společný ani jeden řádek zdrojového kódu. Platí zde podpora paralelního multitaskingu a také podpora víceuživatelského použití.

Ačkoli se to nezdá, tak operační systémy Linux jsou nejrozšířenějším operačním systémem serverů po celém světě. Z této velké rodiny Linuxů se postupně odštěpovaly jednotlivé operační systémy tzv. distribuce. Ty lze volně stáhnout a nabootovat si ho na počítač nebo si ho nejdříve vyzkoušet pomocí tzv. Live CD.

Nejznámějšími linuxovými distribucemi jsou například Ubuntu se svými dalšími odštěpky lišící se hlavně typem grafického rozhraní anebo jejich hlavním použitím a tudíž programovou vybaveností tj. Lubuntu, Kubuntu, Xubuntu, Edubuntu ad., dále jsou to distribuce od firmy Red Hat, což jsou Fedora a Red Hat Enterprise Linux. Dále známou distribucí je Debian, Gentoo, OpenSUSE, Linux Mint a další. Jednotlivé distribuce se od sebe většinou liší programovým vybavením, mírou toho pro jak pokročilé uživatele jsou vhodné (pokud jsou “user friendly“ tak si s nimi poradí každý) a také jsou samozřejmě rozdíly v grafické stránce operačního systému.

Plocha Ubuntu

Posledním dnes velice známým operačním systémem je Mac OS X (Macintosh Operating System X), který byl vyvinut firmou Apple Computer speciálně pro jejich počítače (pokud se je pokusíte nabootovat na jiný počítač, nepovede se vám to, pokud ovšem nezměníte část zdrojového kódu).

Plocha OS S

Na jiných - tablety, smartphony,.. - se dnes také vyskytují operační systémy. Nejrozšířenějším OS na těchto strojích je Android, dále pak iOS a Windows Phone.

Android je operační mobilní systém, který je založený na Linuxu. První verze tohoto OS přišla na trh na zařízení HTC Dream v říjnu 2008. Dodnes vydala několik verzí a aktualizací a ta poslední je z prosince roku 2014 s názvem Lollipop.

iOS je základním operačním systémem většiny výrobků značky Apple (krom Macbooků). Na počátku neměl žádný oficiální název, ale později se pojmenoval iOS (podle iPhone OS). První verze tohoto programu byla vydána na iPhonu v březnu 2008. Nyní je aktuální verze iOS 8.1.3, ale probíhají testování verzí následujících.

Nejmladším operačním mobilním systémem je Windows Phone, který byl poprvé vydán v říjnu 2010. Verze tohoto OS jsou shodné s těmi na počítač tj. Windows 7, 8 a později 10. Zařízení jsou mezi sebou zcela kompatibilní. Ovládání a grafická stránka OS jsou poměrně podobné, což je výhoda oproti ostatním.
