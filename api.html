Aplikační programové rozhraní



API = Application Programming Interface neboli aplikační programové rozhraní je sbírka definicí funkcí a datových struktur, které určují, jak mají mezi sebou programy komunikovat. API představuje abstrakci volané služby. V kontextu operačního systému je API popis rozhraní, pomocí kterého je možné z uživatelského prostoru přistupovat ke službám jádra. Účelem API je lepší přenositelnost a snadná správa kódu. Mezi další použití patří překlad mezi službami vysoké a nízké úrovně. Převádí typy nebo struktury parametrů a také převádí mezi způsoby předávání parametrů (by-value, by-reference).

Ovladač zařízení (device software) je software, které umožňuje operačnímu systému pracovat s konkrétním hardwarem. Operační systém pracuje s periferií pomocí ovladače, do uživatelského prostoru pak služby související s periferií zprostředkovává pomocí API.
